## Schema Changes (Step 1)

**File:** `prisma/schema.prisma` — แก้ไข `CartItem` model

เพิ่ม fields:

```prisma
model CartItem {
  id              String   @id @default(uuid())
  cartId          String
  productId       String
  productName     String              // 🆕 Snapshot: product name at add time
  unitPrice       Decimal  @db.Decimal(10, 2)
  quantity        Int
  subtotal        Decimal  @db.Decimal(10, 2)
  selectedOptions Json?               // 🆕 JSON array of { optionGroupId, optionId, optionName, additionalPrice }
  optionsHash     String?             // 🆕 Hash of selectedOptions for merge detection
  note            String?             // 🆕 Special note for this item
  createdAt       DateTime @default(now())

  cart    Cart    @relation(fields: [cartId], references: [id], onDelete: Cascade)
  product Product @relation(fields: [productId], references: [id])

  @@unique([cartId, productId, optionsHash])  // 🆕 Merge constraint
  @@map("cart_items")
}
```

หลังจากนั้น `prisma generate` + `prisma migrate dev --name add-cart-item-options`

### Zod Schemas (`cart.schema.ts`)
- `selectedOptionSchema` — object: optionGroupId (UUID), optionId (UUID), optionName (string), additionalPrice (number ≥ 0)
- `addToCartBodySchema` — productId (required UUID), quantity (optional int 1-50), selectedOptions (optional array), note (optional string)
- `updateCartItemBodySchema` — quantity (optional int 1-50), note (optional string nullable)
- Route schemas for Swagger

### Service (`cart.service.ts`)
- `getCart(userId)` → findOrCreateByCustomerId → load items with product data → calculate totals
- `addToCart(userId, data)` → validate product exists + active → validate options exist + active + required → check existing (merge) → compute unitPrice (base + options) → create/increment → return cart
- `updateCartItem(userId, itemId, data)` → validate ownership → update quantity/note → recalculate subtotal → return cart
- `removeCartItem(userId, itemId)` → validate ownership → delete item → return cart
- `clearCart(userId)` → delete all items in cart

**Price calculation (server-side only):**
```
unitPrice = product.price + sum(selectedOptions.additionalPrices)
subtotal = unitPrice * quantity
cart total = sum(all item subtotals)
```

**Merge logic:**
1. Hash selectedOptions → SHA-256 hex string
2. Find existing item with same (cartId, productId, optionsHash)
3. If found → increment quantity
4. If not → create new item
5. Wrap in Prisma transaction for atomicity

### Controller (`cart.controller.ts`)
- `getCartHandler` — GET /api/v1/cart
- `addToCartHandler` — POST /api/v1/cart/items
- `updateCartItemHandler` — PATCH /api/v1/cart/items/:id
- `removeCartItemHandler` — DELETE /api/v1/cart/items/:id
- `clearCartHandler` — DELETE /api/v1/cart

### Routes (`cart.routes.ts`)

## Business Rules Summary

1. หนึ่ง customer = หนึ่ง cart (auto-create on first access)
2. Inactive products ไม่สามารถเพิ่มเข้า cart
3. Inactive options ไม่สามารถเลือก
4. Required options ต้องเลือก (Spice Level, Sauce)
5. Server คำนวณ price เสมอ — ไม่ trust client
6. Same product + same options = merge (increment quantity)
7. Different options = separate cart item
8. Cart item เก็บ product name snapshot (กรณี product ถูกแก้ชื่อ)
9. Max quantity per item: 50
10. Cart ไม่มี admin endpoints
